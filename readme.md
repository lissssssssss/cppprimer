例如，题目中提问，如下代码中j的值是多少

	int i = 42;
	int main()
	{
		int j = 100;
		int j = i;
	}
则在程序中实现，并打印j的值

	#include <iostream>
	
	int i = 42;
	int main()
	{
		int j = 100;
		int j = i;
		std::cout << j << std::endl;
		return 0;
	}
